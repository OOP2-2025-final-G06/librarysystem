# アプリ名: oop2_10_sample
> 図書貸出システムアプリ

概要:
> このアプリでは以下のことができます

> トップ画面で各ページにアクセスと入力されたデータの統計確認ができます。

> ユーザー一覧の確認・新規ユーザの追加
> 本の在庫一覧の確認・在庫の追加
> 現在の貸出状況・新規貸し出し登録
